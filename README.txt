# CSS Practice - Balance Sheet

This project is a practical exercise for enhancing your knowledge of CSS, especially focusing on pseudo-selectors.

## Description

The code constructs a balance sheet for the fictional company "AcmeWidgetCorp". The balance sheet spans three years (2019-2021) and outlines their assets, liabilities, and net worth.

The CSS properties used in this project have a special emphasis on:

- Pseudo-selectors
- The `calc()` function
- The `position: sticky` property
- The `white-space` property
- The `border-collapse` property

In addition to understanding these CSS properties and functions, the project is also a good exercise in crafting clean, readable code, which is a crucial skill for any developer.

## How to use?

You can open the HTML file in your web browser to see the layout and styles applied. Feel free to modify the CSS file to experiment and see how the changes affect the output.

# Exercice CSS - Bilan Comptable

Ce projet est un exercice pratique pour améliorer vos connaissances en CSS, en mettant un accent particulier sur les pseudo-sélecteurs.

## Description

Le code construit un bilan comptable pour la société fictive "AcmeWidgetCorp". Le bilan couvre trois ans (2019-2021) et présente leurs actifs, passifs et valeur nette.

Les propriétés CSS utilisées dans ce projet mettent un accent particulier sur :

- Les pseudo-sélecteurs
- La fonction `calc()`
- La propriété `position: sticky`
- La propriété `white-space`
- La propriété `border-collapse`

En plus de comprendre ces propriétés et fonctions CSS, le projet est aussi un bon exercice de rédaction de code propre et lisible, compétence cruciale pour tout développeur.

## Comment l'utiliser?

Vous pouvez ouvrir le fichier HTML dans votre navigateur web pour voir la disposition et les styles appliqués. N'hésitez pas à modifier le fichier CSS pour expérimenter et voir comment les modifications affectent le rendu.
